# SciPy 2026 Starsim-AI Paper

This folder contains the source files for the following manuscript:

> Kerr CC, Rosenfeld K, Lundin J, Stuart R, Abeysuriya R. *Vibes, meet rigor: Evaluating and improving AI performance on complex scientific code*. Proceedings of SciPy 2026.

Note that the manuscript used to live in the [scipy2026_starsim_ai](https://github.com/starsimhub/scipy2026_starsim_ai) repository, and is also duplicated (for submission) in the [scipy_proceedings](https://github.com/starsimhub/scipy_proceedings) repository.

## Structure

- Code for generating the figures is in the `figures` folder.
- The manuscript itself is in the `paper` folder.

*Note*: To regenerate the figures requires the data in the [scipy2026_starsim_exam](https://github.com/starsimhub/scipy2026_starsim_exam) repository. This repository is private to avoid leakage of solutions to subsequent evaluation runs.
